Instant Payment API for E-commerce.

== Description ==

Receive crypto payments from all over the world in your WooCommerce store.

== Installation ==

1. Upload the entire `imsafu-for-woocommerce` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

== Frequently Asked Questions ==

= How to get api key? =

To obtain an API key, simply click 'Join Waitlist' on [imsafu.com](https://imsafu.com) and complete the form provided.

== Screenshots ==

1. Select your preferred payment tokens and network.
2. Transfer the tokens to the provided address on the page.
3. Complete the transfer and wait for confirmation.
4. Your payment has been successfully transferred.
